# Cooking Recipe Web Application

## Overview
This project is a web application for people who enjoy cooking.
It provides simple and quick recipes with a list of ingredients, step-by-step instructions, and a built-in shopping list feature to help users plan grocery purchases.
Users can like recipes, filter by categories, and manage multiple shopping lists for each recipe.

## Features
* Users can create shopping lists from recipes
* Users can like each recipe
* Each shopping list item can be added, marked as completed, or deleted
* Users can create multiple shopping lists per recipe
* Like counts are displayed on each recipe
* Recipes can be sorted from most liked to least liked
* Recipes can be filtered by categories such as Low Calories, Thai, Asian, Western, and Dessert

## Technical Stack
Front-end: HTML, CSS, React

Back-end: Spring Boot

Database: MySQL
